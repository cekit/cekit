About
-----

Container image creation tool.

CEKit helps to build container images from image definition files with strong focus on modularity and code reuse.

Main features
--------------

* :doc:`Building container images </handbook/building/index>` from :doc:`YAML image definitions </descriptor/image>` using many different :doc:`builder engines </handbook/building/builder-engines>`
* :doc:`Integration/unit testing </handbook/testing/index>` of images

I'm new, where to start?
---------------------------

We suggest looking at the :doc:`getting started guide </getting-started/index>`. It's probably the best place to start.
Once get through this tutorial, look at :doc:`handbook </handbook/index>` which describes how things work. Later you may be interested
in the :doc:`guidelines sections </guidelines/index>`.
