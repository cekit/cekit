About
-----

Container image creation tool.

CEKit helps to build container images from image definition files with strong focus on modularity and code reuse.

Features
--------

* :doc:`Building container images </build>` from YAML image definitions using many build engines
* :doc:`Integration/unit testing </testing/index>` of images
